## 5.5 日程汇总页面

展示用户所有训练日程的总览和管理功能。

### Active 训练计划
**计划卡片信息**
- 计划名称：如"Incontinence Training Plan"
- 模式图标：01, 02等编号标识
- 进度条：当前进度可视化
- 进度数字：如"8/24"（已完成/总次数）
- 剩余时间：如"16 sessions left"

<span class="v14-add">**双模式日程**：作为单条 Active 计划展示，1 张计划卡同时显示 2 个 Mode 标记（如 `08 SUI + 09 UUI` 或 `06 Chronic + 07 Acute`），共用进度条 / 颜色 / 进度数字（一日两任务全完成才记 +1）。</span>

**操作功能**
- 点击卡片：查看详细的训练安排
- 编辑功能：修改训练日和提醒时间（不可改模式）
- 删除功能：删除整个训练计划

### Completed 训练计划
**历史计划展示**
- 已完成的训练计划归档显示
- 完成时间和总体评价
- 点击可查看历史训练数据

### 多计划并行
当用户有多个症状模块的训练计划时：
- 不同计划用不同颜色区分
- 粉色：Incontinence
- 橙色：Pain
- 绿色：Hip & Leg Function
- <span class="v14-add">**双模式日程**仍占 1 个颜色槽（按所属模块取色），不会因为绑 2 模式而占 2 槽</span>

### 计划完成处理
训练计划完成后：
- 显示庆祝页面和完成统计
- 引导用户重新评估症状改善情况
- 可选择是否开始新的训练周期

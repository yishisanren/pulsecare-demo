## 编辑日程

### 入口
日程汇总页 → 点击活跃日程卡片

### 可修改项
- **提醒时间**：滚轮选择器
- **训练日安排**：选择每周哪几天训练（Mon-Sun）

### 不可修改项
- 训练模式和时长（需删除后重新评估生成）
- 页面显示锁定图标和说明文案
- <span class="v14-add">**双模式日程**：2 个模式作为整体锁定，**不可单独移除其中一个、不可拆分为两条独立日程**</span>

### 规则
- 已完成日程不可编辑
- 修改训练日后，已完成的训练不受影响
- 右上角 Delete 按钮可删除日程

## 单模块训练计划配置

### 配置项（一页展示）

| 配置项 | 说明 | 交互 |
|--------|------|------|
| 训练模式 | 评估推荐的模式 | 锁定不可修改 |
| 训练周期 | 根据评估结果确定 | **不允许用户选择**，仅展示推荐周期 |
| 训练日期 | 单周视图（Mon-Sun） | 系统根据推荐频率**默认选好日期**，用户可调整，所选日期固定应用到每一周 |
| 提醒时间 | 滚轮选择器 | 默认当前时间，可选 No reminder |

### 日期预选映射表

| 推荐频率 | 默认选择 | 说明 |
|---------|---------|------|
| 1 次/周 | 周三 | 居中分布 |
| 2 次/周 | 周二、周五 | 间隔 3 天 |
| 3 次/周 | 周一、周三、周五 | 隔天训练 |
| 4 次/周 | 周一、周三、周五、周日 | 均匀分布，间隔≥1天 |
| 5 次/周 | 周一至周三、周五、周六 | 周四和周日休息 |
| 每日 1 次 | 全选 | 仅疼痛 Level 3 场景 |

用户可手动调整（点击日期切换选中/取消）。

### 双模式日程触发场景
即使是单模块评估，命中 Mixed 分型也会触发双模式日程：
- 仅评估尿失禁，结果为 Mixed → 触发 08 SUI + 09 UUI 双模式日程
- 仅评估疼痛，结果为 Mixed Acute + Chronic → 触发 06 Chronic + 07 Acute 双模式日程

此时配置项的"训练模式"列锁定为 2 个模式整体（上下排列展示），其他配置项（周期/日期/提醒）按"一日程"共享。

## 模块 3：臀腿功能评估（PGQ，25 题）

### Part A：Activity（Q1-Q20）
"How problematic is it for you because of your pelvic girdle pain to:"

| 选项 | 分值 |
|------|------|
| Not at all | 0 |
| To a small extent | 1 |
| To some extent | 2 |
| To a large extent | 3 |
| Not applicable | 排除不计分 |

**20 个活动项目**：Dress yourself / Stand <10min / Stand >60min / Bend down / Sit <10min / Sit >60min / Walk <10min / Walk >60min / Climb stairs / Do housework / Carry light objects / Carry heavy objects / Get up-sit down / Push shopping cart / Run / Sporting activities / Lie down / Roll over in bed / Normal sex life* / Push with one foot*

**维度标签**：Control / Endurance / Strength（每题有对应维度标签）

### Part B：Symptom（Q21-Q25）
"How much pain/difficulty do you experience:"
- Pain in the morning / Pain in the evening / Legs given way / Do things more slowly / Sleep interrupted

### 计算
- 三个维度各自计算标准化百分比
- 维度得分 ≥ 0.3 → 对应推荐模式

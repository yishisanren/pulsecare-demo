### 计算 / Scoring

**三个维度得分公式**（详见 3.4 评估结果）：

| 维度 | 求和题号 | 标准化公式 | 受限阈值 |
|---|---|---|---|
| Control 控制力 | Q1 + Q16 + Q17 + Q18 + Q20 + Q23（**6 道题**，Q19 已删除）| 实际总得分 / **18** | ≥ 0.3 |
| Endurance 耐力 | Q2 + Q3 + Q5 + Q6 + Q7 + Q8 + Q9 + Q10 + Q12 + Q14 + Q16 + Q24（**12 道题**）| 实际总得分 / **36** | ≥ 0.3 |

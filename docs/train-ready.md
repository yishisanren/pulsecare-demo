## 训练准备页（控制页 · 准备态）

### 功能目的
进入控制页但尚未开始训练时的初始状态。展示当前模式、电流强度起点、设备状态，等待用户点击 **Start** 启动训练。

### 触发时机
- 从日程主页点击当日训练任务卡片进入
- 从设备主页 Training Modes 选中模式后点击进入
- 训练异常停止后用户重新进入控制页（重置为准备态）

### 页面结构

| 区块 | 内容 | 说明 |
|---|---|---|
| 顶部导航 | `<` Training | 返回日程主页 |
| 中央圆形区域 | `--:--` | 计时器占位（未开始） |
| 设备状态 | 🔋67% · 🔵 | 电量 + 蓝牙连接状态 |
| 时长信息 | `Runtime Duration 24mins` + `Ready to Start` | 当前模式总时长 + 准备态文案 |
| Mode 卡片 | `01 Activation · EMS · 24mins` ⓘ | 显示当前模式名称、信号类型、时长 |
| Current Level | `−  0 / 100  +` | 强度调节，**默认 0**，每次训练从 0 开始不做记忆 |
| 主按钮 | **Start**（深色） | 点击后强度自动跳到 1，计时立即开始 → 进入训练中 |

### 关键规则

1. **强度从 0 开始**：每次进入控制页，电流强度强制重置为 0，**不记忆上次值**
2. **点击 Start 行为**：强度自动从 0 跳到 1，计时器立刻启动，按钮切换为 **Pause + Stop**
3. **未开始时不可调节强度**：`+` `−` 按钮在 Start 之前为可视化展示，实际生效从训练中开始
4. **设备未连接**：底部 Start 按钮置灰禁用，提示先连接设备

### 双模式衔接
- 当天双模式日程（如 08 SUI + 09 UUI）：完成第 1 个任务后回到日程主页，点击第 2 个任务卡片重新进入本页


### Mode 卡片 ⓘ 行为
点击右上角 ⓘ 图标 → 弹出**模式选择面板**，可查看 9 种模式说明（仅展示，不允许在此修改当日日程模式；自定义训练除外）。

### 与训练中页面的关系
点击 Start 后立即跳转 → **训练进行中页面**（圆环开始走计时，按钮变为 Pause + Stop）。

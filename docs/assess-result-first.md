### 分型文案 / Classification

主类型由 3IQ 决定（Stress / Urge / Mixed / Other），次级特征由 UDI-6 Q9 / Q10 判定。
Primary type determined by 3IQ; secondary features by UDI-6 Q9 / Q10.

| 3IQ 主类型 | Q9 <2 / Q10 <2 | Q9 <2 / Q10 ≥2 | Q9 ≥2 / Q10 <2 | Q9 ≥2 / Q10 ≥2 |

### 推荐模式 / Recommended Mode

设备模式由 3IQ 主类型独立决定：
Mode selection determined independently by 3IQ primary type.

| 3IQ 主类型 | 推荐模式 / Recommended Mode |
|---|---|
| Stress | Mode 8 · SUI 模式 / SUI Mode (Stress Urinary Incontinence) |
| Urge | Mode 9 · UUI 模式 / UUI Mode (Urge Urinary Incontinence) |
| Mixed | Mode 8 + Mode 9 交替 / SUI Mode + UUI Mode alternating |
| Other | 不推荐模式，建议转诊 / Not recommended; refer to specialist |

### 评估结果 · 疼痛（非首次）

### 趋势图 / Trend Chart

- 始终展示，最多显示最近 7 次评估 / Always visible, shows up to 7 most recent assessments
- 纵轴为百分比，横轴为评估次序 / Y-axis: percentage, X-axis: assessment index
- 1 次数据显示 1 个点 / With 1 data point, show a single point


---

### 分型文案 / Classification (Q2 决定)

| Q2 描述词 | 中文类型（PRD 原文） | English | 推荐模式 |
|-----------|----------------------|---------|----------|
| Cramping / Tightening / Spasm | 急性肌肉痛 | Acute Muscular Pain | Mode 7 |
| Sharp / Stabbing / Shooting | 急性组织痛 | Acute Tissue Pain | Mode 7 |
| Aching / Dull / Heavy | 慢性伤害感受痛 | Chronic Nociceptive Pain | Mode 6 |
| Burning / Tingling / Electric | 慢性神经痛 | Chronic Neuropathic Pain | Mode 6 |
| Pain with Light Touch | 慢性神经敏化 | Chronic Neural Sensitization | Mode 6 |
| Comes and Goes | 混合型 | Mixed Type | Mode 6 + Mode 7 |

### 评分等级 / Score Level (NRS 0-10)

- NRS 0 → **Level 1（无症状，不训练）** / Level 1 (No symptoms, no training)
- NRS 1-3 → **Level 2**（轻度疼痛 / Mild pain）
- NRS 4-6 → **Level 3**（中度疼痛 / Moderate pain）
- NRS 7-10 → **Level 4**（重度疼痛 / Severe pain）

### 安全门控 / Safety Gate

Q1 选了风险项 → 不推荐模式 + 警告文案 / If any risk item is selected in Q1 → no training recommended + warning

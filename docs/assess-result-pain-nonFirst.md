### 分型文案 / Classification (Q2 × Q3)

详见首次评估结果页完整映射表。非首次评估沿用相同逻辑。
See first-assessment page for full Q2 × Q3 mapping. Follow-up uses the same logic.

| Q2 类型 | 示例文案 / Example |
|---|---|
| Acute / Muscle-related | 产后早期肌肉紧张或痉挛，常见且多可自行缓解。<br/>Early postpartum muscle tension or spasm; common and often self-limiting. |
| Acute Nociceptive | 产后早期急性伤害感受痛，常与局部组织损伤相关。<br/>Early postpartum acute nociceptive pain, often related to local tissue injury. |
| Chronic Nociceptive | 长期慢性伤害感受痛，建议结合训练与专科评估。<br/>Long-term chronic nociceptive pain; combine training with specialist evaluation. |
| Chronic Neuropathic | 慢性神经痛，建议结合专科治疗。<br/>Chronic neuropathic pain; combine with specialist treatment. |
| Chronic Mixed | 慢性混合型疼痛，建议多学科评估。<br/>Chronic mixed pain; multidisciplinary evaluation recommended. |

### 严重程度 + 训练频次 / Severity + Training Frequency (NRS)

| NRS | 严重程度 / Severity | 推荐模式 / Mode | 频次 |
|---|---|---|---|
| 0 | 无症状，不训练 / No symptoms; no training needed | — | 无需 TENS / No TENS recommended |
| 1–3 | 轻度疼痛 / Mild pain | Mode 6 或 Mode 7（按类型）| 3次/周 × 2周 |
| 4–6 | 中度疼痛 / Moderate pain | Mode 6 或 Mode 7（按类型）| 5次/周 × 3周 |
| 7–10 | 重度疼痛 / Severe pain | Mode 6 或 Mode 7 + 转诊建议 | 5次/周 × 4周 |

**Mode 命名 / Mode Names**：
- Mode 6 · 镇痛舒缓 / Chronic Pain Relief Mode
- Mode 7 · 止痛快速缓解 / Acute Relief Mode

## 请假后日程主页

### 跳过的任务显示
- 任务卡片变为半透明（opacity 0.5）
- 模式名称加删除线
- 状态文字显示橙色 **Skipped**
- 其他未跳过的任务保持正常

### 撤销操作
- 点击已跳过的任务卡片 → 弹出撤销确认弹窗
- 确认后任务恢复为 Pending 状态

### 撤销时限
| 场景 | 时限 |
|------|------|
| 当天请假 | 当天 00:00 前可撤销 |
| 未来日期请假 | 该日期到来前可撤销 |
| 已过期日期 | 不可撤销 |

## 5.4 训练计划预览

展示用户配置完成的训练计划详情，供用户最终确认。

### 单模块预览
**计划摘要卡片**
- 训练周期：如"6 weeks"
- 训练模式：如"Activation"
- 总训练次数：如"24 sessions"
- 提醒时间：如"8:30 PM daily"

**周计划展示**
- 按周分组显示：Week 1, Week 2...
- **仅展示训练日**，不显示休息日
- 训练日：显示星期+日期、模式名称、训练时长
- **双模式日程**：训练日同时显示 2 个 Mode 编号（如 `06 Chronic + 07 Acute` 或 `08 SUI + 09 UUI`），共享同一行

### 多模块预览（Tab版）
**顶部Tab栏**
- 可切换查看不同模块的训练计划
- Tab标签：Incontinence / Pain / Hip & Leg Function

**计划摘要**
- 每个Tab显示对应模块的计划摘要
- 用颜色区分不同模块：粉色/橙色/绿色

**合并日历视图**
- 单日有多个**模块**训练时（不同日程），用不同颜色圆点标识
- 单日**双模式日程**只占 1 个颜色槽，日历仍显示 1 个圆点（同色），任务区当天展开 2 张独立卡片
- 训练日显示多个模式编号

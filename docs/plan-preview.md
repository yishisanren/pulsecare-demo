## 5.4 训练计划预览

展示用户配置完成的训练计划详情，供用户最终确认。

### 单模块预览
**计划摘要卡片**
- 训练周期：如"6 weeks"
- 训练模式：如"Activation"
- 总训练次数：如"24 sessions"
- 提醒时间：如"8:30 PM daily"

**周计划展示**
- 按周分组显示：Week 1, Week 2...
- **仅展示训练日**，不显示休息日
- 训练日：显示星期+日期、模式名称、训练时长

### 多模块预览（Tab版）
**顶部Tab栏**
- 可切换查看不同模块的训练计划
- Tab标签：Incontinence / Pain / Hip & Leg Function

**计划摘要**
- 每个Tab显示对应模块的计划摘要
- 用颜色区分不同模块：粉色/橙色/绿色

**合并日历视图**
- 单日有多个模块训练时，用不同颜色圆点标识
- 训练日显示多个模式编号

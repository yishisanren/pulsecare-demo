## 2.3 设备主页

设备主页是用户与产品交互的核心界面，根据设备连接状态、日程状态和评估状态展现不同功能入口。

### 2.3.1 页面构成

设备主页由以下模块构成：
- **Device Connection Area**：显示设备连接状态和电量信息
- **Assessment Module**：问卷评估相关入口和状态
- **Training Schedule**：今日训练任务和日程概览
- **Training Modes**：自定义训练模式选择
- **Data Records**：训练记录和评估记录入口

### 2.3.2 状态变化

根据不同状态组合，主页呈现不同布局：

**已连接 + 有训练任务**
- 设备区域：显示已连接状态、电量百分比
- 日程卡片：今日训练任务，显示模式、时长、开始按钮
- 其他模块：正常展示

**已连接 + 今日已完成**
- 日程卡片：显示"今日训练已完成"状态
- 设备区域：保持连接显示

**设备未连接**
- 设备区域：灰色设备图标 + "Connect Device"按钮
- 其他功能：正常可用但训练相关功能引导连接设备

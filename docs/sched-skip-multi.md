## 跳过范围选择（多任务场景）

### 触发条件
当天有 **多条** 未完成训练任务时，点击 Skip 后弹出。<span class="v14-change">两类场景都会触发：</span>
- <span class="v14-add">多模块多任务（不同日程当天叠加）</span>
- <span class="v14-add">**双模式日程内 2 任务**（同一日程当天展开的 2 张卡片）</span>

### 弹窗选项
- **"Skip this session"** → 仅跳过本条任务，标记为 Skipped
- **"Skip all today's sessions"** → 跳过当天全部未完成任务（已完成不受影响）
- **"Cancel"** → 返回

### 规则
- 当天仅 1 条未完成任务时直接跳过，不弹此选择
- 跳过的任务标记为 Skipped，不重新安排到其他日期
- 日程周期不变，训练总次数相应减少
- <span class="v14-add">**双模式日程**：只 Skip 1 个任务时该日程当天**不算请假**；2 个都 Skip 才算该日程当天请假</span>
